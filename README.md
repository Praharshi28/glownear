# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

# 🌸 GlowNear — Mumbai's AI Beauty Marketplace

> Mumbai's #1 AI-powered beauty salon booking platform built for SuperXgen AI Startup Buildathon 2026

## 🚀 Live Demo
[glownear.vercel.app](https://glownear.vercel.app)

## 📱 Features

### Core Features
- 🏪 **9+ Mumbai Salons** — Browse, filter and compare top salons
- 📅 **Smart Booking** — Book appointments with occasion-based flow
- 💳 **Payment Methods** — UPI, Card, Net Banking, Cash on Arrival
- ⭐ **Reviews & Ratings** — Verified customer reviews with helpful votes

### AI Features
- 🤖 **AI Chat Assistant** — Beauty recommendations in English & Hindi
- 📸 **Live Camera Beauty AI** — Real-time face analysis & makeup suggestions
- ✨ **Glow Score™** — Unique AI-powered salon rating system
- 🎨 **Vision Board** — AI picks your perfect beauty style

### Planning Tools
- 💰 **Budget Planner** — Plan beauty routine within your budget
- 📅 **Occasion Planner** — Wedding/Party beauty countdown timeline
- ⚖️ **Salon Comparison** — Compare 3 salons side by side
- 🎲 **Surprise Me** — AI picks perfect salon + service instantly

### More Features
- 🏠 **Home Services** — Book beauty at your doorstep
- 🎬 **Makeup Tutorials** — Step-by-step guides for every occasion
- 👑 **Admin Dashboard** — Full salon management system
- 🔐 **User Auth** — Login/Signup with Supabase
- 📊 **My Bookings** — Track all appointments & spending

## 🛠️ Tech Stack
